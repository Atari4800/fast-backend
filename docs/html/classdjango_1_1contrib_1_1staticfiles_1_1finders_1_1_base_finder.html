Given a relative file path, find an absolute file path.

If the ``all`` parameter is False (default) return only the first found
file path; if True, return a list of all found files paths.

Reimplemented in django.contrib.staticfiles.finders.FileSystemFinder, django.contrib.staticfiles.finders.AppDirectoriesFinder, and django.contrib.staticfiles.finders.BaseStorageFinder.
