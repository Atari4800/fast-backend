Used to support list values in HTML forms.
Supports lists of primitives and/or dictionaries.

* List of primitives.

{
    '[0]': 'abc',
    '[1]': 'def',
    '[2]': 'hij'
}
    -->
[
    'abc',
    'def',
    'hij'
]

* List of dictionaries.

{
    '[0]foo': 'abc',
    '[0]bar': 'def',
    '[1]foo': 'hij',
    '[1]bar': 'klm',
}
    -->
[
    {'foo': 'abc', 'bar': 'def'},
    {'foo': 'hij', 'bar': 'klm'}
]

:returns a list of objects, or the value specified in ``default`` if the list is empty
