Detailed Description

Iterative access to coordinate tuples from the parent geometry's coordinate
sequence.

Example:

  >>> from shapely.wkt import loads
  >>> g = loads('POINT (0.0 0.0)')
  >>> list(g.coords)
  [(0.0, 0.0)]
