\doxysubsection{Detailed Description}
\begin{DoxyVerb}A zero dimensional feature

A point has zero length and zero area.

Attributes
----------
x, y, z : float
    Coordinate values

Example
-------
  >>> p = Point(1.0, -1.0)
  >>> print(p)
  POINT (1.0000000000000000 -1.0000000000000000)
  >>> p.y
  -1.0
  >>> p.x
  1.0
\end{DoxyVerb}
